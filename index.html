Smooth Products

A simple, lightweight, and responsive jQuery product image viewer by Kevin Thornbloom.

Features

  • Lightweight script
  • Animates to different image heights (see fourth item in demo)
  • Click to zoom & pan image
  • Responsive image sizes (resize this page)
  • Indicator for thumbnail currently open

Usage

  1. Add smoothproducts.js and smoothproducts.css to your page
  2. Create a div with a class of "sp-wrap".
  3. Place medium sized images within "sp-wrap" and link them to the full size image
  4. Smoothproducts takes care of the rest!
